## Description 
Message broker RabbitMQ to send email

Database MongoDB to persist logs

<i>PS: Persistence layer has been implemented here for didactic use only.</i>

## Installation

```bash
$ npm install
```

## Running the app

```bash
# environment
$ npm run start

# development
$ npm run dev

# watch mode
$ npm run debug

# production mode
$ npm run prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
